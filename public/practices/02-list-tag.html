List Tag

ภาษา HTML จะมี TAG สำหรับการแสดงผลที่เป็นรายการ(list) อยู่ 3 ประเภทคือ

  1. รายการแยยใช้สัญลักษณ์
  2. รายการแบบใช้ตัวเลข
  3. รายการแบบหัวข้อและเนื้อหา

รายการแบบใช้สัญลักษณ์

จะใช้ TAG ชื่อ ul เป็น TAG หลัก และเราจะใส่ TAG ที่ชื่อว่า li ข้างใน TAG ul อีกที
ดย TAG li 1 TAG จพสำหรับ 1 รายการ ถ้ามี 3 รายการก็หมายความว่าจะมี TAG li อยู่ใน TAG ui 3 TAG

ตัวอย่างเช่น

ภาควิชาเทคโนโลยีมีเดีย

  • อจิทัลมีเดียเทคโนโลยี
  • พัฒนาเกมส์
  • ชีวการแพทย์

รายการแบบใช้ตัวเลข

จะมีรูปแบบคล้ายกับ รายการแบบสัญลักษณ์โดยเราจะเปลียน TAG หลักจาก ul เป็น ol แค่นั้น ส่วนภายใน TAG ol ก้อจะมี TAG li เป็นตัวกำหนดรายการตามเดิม

ตัวอย่างเช่น

เหตุผลหลักที่ทำให้การเรียนเขียนโปรแกรมเป็นไปอย่างยากลำบาก

  1. อ่านภาษาอังกฤษไม่ออก
  2. จำคำศัพท์ไม่ได้
  3. ไม่จดคำศัพท์เฉพาะ

รายการแบบหัวข้อและเนื้อหา

รูปแบบนี้จะแตกต่างจากสองแบบแรกอยู่พอสมควรคือ TAG หลักคือ dl แต่ภายในจะไม่ใช้ TAG li
โดยจะเปลี่ยนไปใช้ 2 TAG ด้วยกันคือ dt ใช้สำหรับรายการที่เป็นหัวข้อ และ dd ใช้สำหรับรายการที่เป็นเนื้อหา
และแน่นอนว่า ทั้ง 2 TAG นี้จะต้องอยู่ใน TAG dl

ตัวอย่างเช่น

ชื่อมหาวิทยาลัย
มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าธนบุรี
ชื่อสาขาวิชา
สาขาวิชาเทคโนโลยีมีเดีย

List ซ้อน List

ในบางครั้งเราสามารถมี List ย่อยอยู่ใน List ใหญ่ได้ โดยวิธีการคือ ให้ทำ TAG หลักของ (ul,li,dl)ไปใส่ไว้ใน TAG li จากนั้นก็ใส่ TAG li ใน TAG หลัก
ที่พึ่งเพิ่มเข้าไป ตามปกติ
